# CS50P Python Coursework

This repository contains my Python coursework from Harvard's CS50P: Introduction to Programming with Python.

The projects in this repository demonstrate foundational Python programming skills including variables, conditionals, loops, functions, string manipulation, input validation, dictionaries, file organization, and problem-solving using clean, readable code.

## Topics Covered

* Python syntax and program structure
* Functions and modular design
* Conditional logic
* Loops and iteration
* String processing
* Dictionaries and data handling
* Input validation
* Basic testing and debugging
* Command-line program execution

## Project Folders

* `bank` - Conditional logic and string handling
* `camel` - String conversion and formatting
* `coke` - Looping and user input validation
* `deep` - Simple text comparison
* `einstein` - Numeric calculation
* `faces` - String replacement
* `fuel` - Fractions, percentages, and error handling
* `grocery` - Dictionary-based item counting
* `indoor` - String normalization
* `interpreter` - Basic arithmetic expression handling
* `meal` - Time conversion and conditional logic
* `nutrition` - Dictionary lookup
* `outdated` - Date parsing and formatting
* `plates` - Input validation rules
* `playback` - String formatting
* `taqueria` - Menu lookup and running totals
* `tip` - Numeric conversion and calculation
* `twttr` - String filtering
